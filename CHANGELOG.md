# 更新日志

本文件用于记录导表工具后续所有版本的更新动态。

## v0.4.0

发布时间：2026-04-24

更新内容：

- 将数组字段的填写格式从竖线分隔（`101|102`）更改为 JSON 风格的方括号与逗号组合（`[101, 102]`），提升人眼可读性
- 移除了旧版的竖线分隔支持，现强制要求数组字段使用 `[ ... ]` 格式
- 更新了相关文档与项目首页的 README
- 更新版本至 `v0.4.0`

## v0.3.0

发布时间：2026-04-24

更新内容：

- 按照策划需求重新调整表头顺序：第 1 行名称、第 2 行类型、第 3 行默认值、第 4 行英文标识、第 5 行中文标识、第 6 行起为内容
- 新增默认值功能：如果第 3 行的某列有默认值，且数据行该列为空，则自动使用该默认值
- 新增跳过行功能：如果某行的第一列以 `#` 开头，则忽略整行不导出
- 更新版本至 `v0.3.0` 并补充对应文档

## v0.2.0

发布时间：2026-04-23

更新内容：

- 支持策划习惯的新表头顺序：第 1 行填写类型，第 2 行填写字段名，第 3 行填写中文注释
- 兼容旧表头顺序：第 1 行中文注释，第 2 行字段名，第 3 行类型
- 更新策划使用说明，补充 `Sheet` 命名规则、重名覆盖后果和新表头规范
- 发布第二版安装包，归档到 `Releases/v0.2.0`
- 新增统一维护的更新日志文件，后续版本继续追加到本文件

## v0.1.0

发布时间：2026-04-23

更新内容：

- 完成首版导表工具开发
- 支持扫描 `Excel` 目录下所有 `.xlsx` 文件
- 支持导出 JSON、C# 配置类、`manifest.json` 和 `ConfigTables.cs`
- 支持中文错误汇总、主键校验、重复 `id` 校验
- 完成首版打包发布，归档到 `Releases/v0.1.0`
