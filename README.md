# Unity Excel2Json Data Import Tool

一款轻量、开箱即用的 Excel 导表工具。无需配置复杂的 Python 环境，支持一键将策划编写的 Excel `.xlsx` 格式配置表转化为 Unity 友好的 JSON 数据文件和对应的 C# 强类型配置类。

## 🌟 核心特性

- **零依赖一键运行**：打包为独立的 Windows 可执行文件 (`.exe`)，无需在本地安装 Python 即可使用。
- **类型强绑定**：支持导出基础类型及数组类型，配套生成 C# Config 类供 Unity `JsonUtility` 反序列化。
- **支持默认值逻辑**：在 Excel 中配置列级默认值，留空单元格将自动继承默认值，精简表内冗余信息。
- **友好的数组格式**：抛弃难懂的 `|` 分隔符，使用类似 JSON 的 `[val1, val2]` 格式来书写数组。
- **屏蔽行支持**：使用 `#` 开头的主键行会被工具自动忽略，方便策划书写测试数据或废弃数据。

## 📂 推荐目录结构

```text
导表工具目录/
├── Excel2Json.exe          # 工具可执行文件
├── gen_data.bat            # 启动脚本
├── 策划使用说明.md         # 给策划的操作指南
└── Excel/                  # 配置表源文件目录
    ├── Item.xlsx
    └── Skill.xlsx
```

## 📝 填表规范

第一行到第五行为表头控制区域，具体规范如下：

| 行号 | 内容 | 说明 |
|------|------|------|
| **1** | 名称 | 表名描述（工具不解析，可写任意内容） |
| **2** | 类型 | 数据类型，例如 `int`, `string`, `array_int` 等 |
| **3** | 默认值 | 若某列配置了默认值，则下方留空数据行将使用该值 |
| **4** | 英文标识 | 对应输出 JSON 和 C# 中的字段名称 |
| **5** | 中文标识 | 给策划看的字段中文注释，导表时跳过 |
| **6起** | 数据 | 实际配置内容。首列带 `#` 会被整行忽略 |

## 🚀 构建与打包

如果你是程序人员，希望修改逻辑或重新打包最新版：

1. 确保安装了 Python 3.9+ 环境。
2. 安装依赖：
   ```bash
   pip install -r requirements-build.txt
   ```
3. 运行构建脚本：
   ```bash
   python build_release.py
   ```
4. 构建结果将自动归档至 `Releases/vX.X.X/` 目录下。
